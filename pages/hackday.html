Firefox OS Hackday

You need to follow the next steps to be prepared for our second Open Web Device hackday!

Step 1

You should have Linux or Mac. And you need the make and git tools. Check how you can install it and have it on your path to be used in your terminal app.

We don't know if there is an easy way to get make working on Windows. So it's better if you can create a VM with some Linux distro you are confortable with.

If you really REALLY like Windows we heard that Cygwin can be useful to get make running. Although we haven't tested yet, and we can't support you.

Step 2

Install and know how to use your code editor. A plain text editor (vi or emacs), SublimeText or whatever you like, but, please, master it! We know that master a tool is a good performance booster.

Step 3

If you want to develop in your own computer (which is good if you don't have a phone or want to test quickly), you will need the Nightly version of Firefox. Just browse to this page and download your version. You can use different profiles with the same Firefox installation or with different versions. Check this page if you want to separate your profiles.

Step 4

Create a hackday directory wherever you want. Like in your home path: ~/hackday/, and enter it.

Download Gaia: git clone https://github.com/mozilla-b2g/gaia.git. It will take some time, since it's around ~250MB.

Make Gaia: You might think that Gaia shouldn't be compiled, and it isn't! But there are some things that need to be maked to create a profile. Since Gaia is not just webapps, but "special" webapps (need permissions to do some things that are not allowed to normal apps), this step is neccesary. This will download xulrunner (between 50 and 250MB) and make some interesting things.

So, just type DEBUG=1 make and watch the dance. Once finished, you will have a Firefox profile that you can open to test Gaia. In fact, you will have a message like this:
Profile Ready: please run [b2g|firefox] -profile /YOUR/PATH/TO/gaia/profile

Then, you can open a specific application, like:
firefox -profile /YOUR/PATH/TO/gaia/profile http://contacts.gaiamobile.org:8080/
where firefox is the path to your Firefox binary.

This will open that Firefox profile with the app that you have selected. You can play with it (and the Firefox developer tools).

Step 5

Download the AndroidSDK

This step is mandatory if you are going to develop with a phone, and want to see your awesome app in a real phone.
Go to Android SDK page and download accordingly to your Operating System.

Follow the steps, and install the "Android SDK Platform-tools" which installs adb, the neccesary binary to push files to the device.

Note that if you are using Linux, you have two options, to use sudo while working with adb, or adding udev rules. We suggest using the second way, if so, follow this guide.
