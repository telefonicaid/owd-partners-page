Adapting your application

Do you have an HTML5 application? Check this steps to ensure it's ready for the Open Web Device

If you have already made the effort to create your HTML5 application and adapted it to be compatible with mobile devices this is your lucky day!

You did almost all the hard work, but if you want to be sure your app will be ready for the Open Web Device follow the next steps and you'll be ready.

Step 1

Check your app using Firefox Nightly. Open Web Device is using the Mozilla technology that is the same across the different products/browsers. This vesion contains the latest updates and changes. This is the closet version that you can run on your desktop and has almost the same specifications that you can find in the Open Web Device.

You can download Firefox Nightly here for the common operating systems.

Also you can make Firefox Nightly to display your app with the size you'll find in the Open Web Device, go to the menu Tools > Web Developer > Responsive Mode and select the resolution 320x480.

Step 2

Did you create your application manifest?

Check the documentation in this page, here you'll find the information to define your app for the Open Web Device.

Step 3

Be offline aware.

We show previously in the documentation how to be able to run your app in offline mode. Remember in a mobile phone connectivity is not always present!

Step 4

Interested in the new APIs that the phone offers?

Yes! There are several new APIs related to phone specific matters. There is a huge list of new features, are you interested on them? Go and check the Mozilla wiki, here you'll find all the new APIs available in the Open Web Device.
