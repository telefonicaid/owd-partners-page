HTML5 technologies

What's behind this famous word?

So, seriously what's HTML5?

What everyone use to call HTML5 is not just a single technology but the combination of: HTML (new spec) + Javascript + CSS3

Each of them are old friends, but the new capabilities that they offer combined will increase your development success to an incredible level of awesomeness.

New things to try

Ok, let's take a quick look to the list of new goodies that our old friends HTML5, Javascript and CSS have brought to HTML5

Offline/Storage

We already talk about this important feature for B2G in this page. Is quite important for web apps and you should always have it in mind.

Besides the information here, you could learn more here: what means going offline?

Realtime communication

You could be interested in knowing how to make the browser perform to the limit, check how to work with WebWorkers, make them communicate, or directly talk to the web via WebSockets.

Deeper integration with the OS

Go and play with new APIs like Drag and Drop, the Filesystem API, knowing user's location via the Geolocation API, or just know the device orientation of the user device.

New semantic tags

New set of tags that add better semantics, specific markups for applications, descriptive links relations, microdata, accessibility tags (ARIA), those are some of the new additions to the markup. Go and check all the new tags.

New form types

HTML5 is adding new form types and attributes that will help improve the semantic of your app, but also help the browser/OS to render a better experience, like specific inputs for numbers, date, etc. There is a lot of new features, dive into them!

New multimedia capabilities

As you should know, one of the top features is the native implementation for audio and video features.

But also you could find interesting playing with a canvas where you can draw and even perform 3D renders, check it out here!

What about the new CSS3 features?

Oh boy!

So many new things in CSS3, that we would need a book to talk about them, like CSS3 selectors, WebFonts,columns,opacity,rounded corners,text shadows,gradients,flexbox,transtions, transforms and animations

But there is more, much more waiting for you to play with!
